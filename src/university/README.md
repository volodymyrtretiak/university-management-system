# ==========================================
# UNIVERSITY MANAGEMENT SYSTEM
# ==========================================

## Опис проєкту

University Management System — консольна система управління університетом, розроблена мовою Java.

Проєкт реалізує функціонал роботи зі студентами, викладачами, курсами, зарахуваннями, оцінками та розрахунком GPA.

---

## Мета проєкту

Демонстрація практичного використання:

- Об'єктно-орієнтованого програмування (OOP)
- Наслідування (Inheritance)
- Інкапсуляції (Encapsulation)
- Поліморфізму (Polymorphism)
- Enum
- Interface
- Exception Handling
- Роботи з масивами
- Організації багатофайлового Java-проєкту

## Використані принципи ООП

| Принцип | Реалізація |
|----------|------------|
| Inheritance | Student та Teacher успадковують Person |
| Encapsulation | Усі поля private |
| Polymorphism | Перевизначення toString(), equals(), hashCode() |
| Abstraction | Виділення сервісних класів |

## Валідація даних

У проєкті реалізовано перевірку:

- коректності ID;
- email адрес;
- порожніх рядків;
- null значень;
- некоректних оцінок;
- некоректних статусів.

Для обробки помилок використовується:

```java
IllegalArgumentException
```
